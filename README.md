# My Python App

## Overview
This project is a Python application designed to provide a user-friendly interface for managing various tasks. It includes features for handling drafts, processing subtitles, and managing audio settings.

## Project Structure
```
my-python-app
├── config
│   ├── default_profile.json
│   └── settings.json
├── core
│   ├── __init__.py
│   ├── click_engine.py
│   ├── cleaner.py
│   ├── draft_manager.py
│   ├── subtitle_engine.py
├── ui
│   ├── __init__.py
│   ├── components.py
│   ├── main_window.py
│   ├── step1_project.py
│   ├── step2_style.py
│   ├── step3_audio.py
│   └── step4_execute.py
├── utils
│   ├── __init__.py
│   └── helpers.py
├── app.py
├── requirements.txt
└── README.md
```

## Installation
To install the required dependencies, run the following command:

```
pip install -r requirements.txt
```

## Usage
To start the application, run:

```
python app.py
```

Follow the on-screen instructions to navigate through the application.
